• git config --global user.name blabla - необхідно при першому вході в GIT
  • git config --global user.email blabal@bla - необхідно при першому вході в GIT
  • git init - команда, яка створює локальний репозиторій в папці (необхідно прописати відразу після створення папки на компі)
  • git status - показує статус змін в репозиторію
  • git add - вказує дані, які будуть закомічені
  • git commit - фіксує зміни
  • git log
  • git checkout - для переключення між гілками
  • git branch - для створення гілок
  • git push - для відправки змін на веб.репозиторій
  • git pull - для загрузки змін з веб.репозиторію
  • git merge - команда, яка зліплює до купи дві гілки
  • git clone - команда, яка скачує на комп репозиторій
  • git revert
  • git reset
  • git rebase
  • git cherry-pick
  • git stash
